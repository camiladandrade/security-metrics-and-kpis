# Security Metrics & KPIs (GRC • VM • IR)

Repositório com **catálogo de KPIs/KRIs**, **dicionário operacional de métricas**, templates de **report executivo** e exemplos (fictícios) para demonstrar como estruturar **métricas de Segurança** para liderança.

✅ **Comece por aqui:** `./00-executive-pack/README.md`

---

## 🎯 Objetivo

Ajudar a responder, com dados:
- Estamos reduzindo risco?
- O que piorou/melhorou e por quê?
- Onde investir primeiro?
- Quais áreas estão fora do SLA (ou acumulando “aging” crítico)?
- O que precisa de decisão executiva?
